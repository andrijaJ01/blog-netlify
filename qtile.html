How to set up Qtile

by Andrija JovanovicApril 23rd 2021

Above you can see couple examples of what kind of customization can be acomplished with qtile, but what exactly is qtile? It is simple window manager written and configured in  python, so it has the potential to be great learning and practice material for begginer python developers.

First we are going to install  requirements, this is assuming that you have installed archlinux or arch based Linux distribution:

sudo pacman -S python python-pip qtile rofi python-pywal ttf-font-awesome
